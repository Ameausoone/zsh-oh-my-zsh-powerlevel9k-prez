== Installation
image::{imagedir}/sfeir_fond_big.png[background, size=cover]

[source]
MacOS
----
$ brew install zsh zsh-completions
$ chsh -s /usr/local/bin/zsh
----

[source]
Ubuntu
----
$ apt-get install zsh
$ chsh -s `which zsh`
----

== Usage

[source]
----
$ ~/.zshrc # Default file for configuration
----
